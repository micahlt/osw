System Requirements

To run a stable instance of OSW (Operating System on the Web) you need one of these browsers:

  • Chrome 80+
  • Firefox (Quantum) 71+
  • Edge (Chromium-based) 80+
And you'll need to be running one of the following host operating systems:
  • Windows Vista/7/8/8.1/10
  • MacOS (OSX)
  • Linux x86-64
We also recommend that you have at least 4GB of RAM and at least 512MB of storage to have the best experience with OSW.
